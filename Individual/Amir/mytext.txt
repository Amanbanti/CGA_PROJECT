## Lesson 15: Conditionals

Boolean Expressions

Boolean expressions evaluate to true or false. Example: Comparing properties of two sprites.

    sprite1.y == sprite2.y: true (y-values are equal)
    sprite1.x > sprite2.x: false (sprite1's x is less than sprite2's x)
    sprite1.x < sprite2.y: true

Conditional Logic Examples

    Changing Fruit Animation
    When the fruit's scale exceeds 2, its animation changes from an apple to a pear.

    fruit.setAnimation("apple");
    fruit.scale = 0.1;

      background("white");
      fruit.scale = fruit.scale + 0.01;

      if (fruit.scale > 2) {
          fruit.setAnimation("pear");
      }

      drawSprites();
    }

    Race Car Finish Line
    The program checks if the race car has crossed the finish line (raceCar.x < finishLine.x) and displays the boolean result dynamically as the car moves.

      text(raceCar.x < finishLine.x, 100, 100);

    }

    Magic Dinosaur Transformation
    The dinosaur turns into a pterodactyl when it reaches the sky (dinosaur.y < 200).
